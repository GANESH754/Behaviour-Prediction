# Behaviour-Prediction

Psyche Atlas is an AI-based personality analysis system that evaluates
user behaviour using questionnaire-based scoring.

## Features
- Personality analysis
- Trait scoring algorithm
- Radar chart visualization
- Behaviour indicators
- Confidence level

## Technologies Used
HTML
CSS
JavaScript

## How it Works
Users answer 10+ questions. The system calculates weighted trait scores
and displays the dominant personality profile.
